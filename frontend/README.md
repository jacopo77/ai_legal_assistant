Paralegal Assistant - Frontend (Next.js)
========================================

Quickstart
----------

1) Set backend URL in `.env.local`:

```
NEXT_PUBLIC_BACKEND_URL=http://127.0.0.1:8000
```

2) Install and run:

```
npm install
npm run dev
```

Open http://localhost:3000

Usage
-----
- Type a question and optional country; the page streams the answer from the backend.
